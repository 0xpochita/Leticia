# Leticia — Demo Video Script

## Overview untuk Editor

Halo editor! Sebelum mulai, ini penjelasan singkat tentang project ini agar kamu paham context-nya.

### Apa itu Leticia?

Leticia adalah aplikasi keuangan (DeFi) yang memungkinkan orang **menyimpan aset crypto dan mendapat bunga tetap** — mirip deposito di bank, tapi di blockchain.

Bayangkan seperti ini: kamu taruh uang di bank, bank bilang "kamu dapat bunga 10% per tahun, dijamin." Nah Leticia melakukan hal yang sama, tapi untuk aset crypto di blockchain Initia.

### Kenapa ini penting?

Di dunia crypto, kalau kamu staking (menyimpan) aset, bunga yang kamu dapat **berubah-ubah setiap hari** — kadang 15%, kadang 5%, tidak pernah pasti. Ini membuat orang susah merencanakan keuangan mereka.

Leticia menyelesaikan masalah ini: **kamu deposit, rate-nya dikunci, kamu tahu persis berapa yang kamu dapat di akhir.**

### Istilah yang perlu kamu tahu

| Istilah | Arti Simpelnya |
|---|---|
| **Deposit** | Menyetor / menaruh aset ke dalam aplikasi |
| **Fixed Yield** | Bunga tetap yang dijamin — tidak berubah setelah kamu deposit |
| **APR** | Annual Percentage Rate — bunga per tahun (misal 10% APR = kalau deposit 1000, dapat 100 dalam setahun) |
| **Wallet** | Dompet digital untuk menyimpan crypto (seperti rekening bank digital) |
| **Maturity** | Tanggal jatuh tempo — waktu dimana kamu bisa ambil uang + bunga |
| **Token** | Aset digital / mata uang crypto (INIT, sINIT, USDe adalah nama-nama token) |
| **Faucet** | Tempat untuk minta token gratis (untuk testing, seperti uang monopoli) |
| **Smart Contract** | Program otomatis di blockchain yang menjalankan aturan — seperti mesin ATM yang tidak perlu pegawai bank |

### Alur cerita video ini

```
1. HOOK      → Sampaikan masalah: bunga crypto tidak pasti
2. PROBLEM   → Jelaskan kenapa ini masalah besar
3. SOLUTION  → Perkenalkan Leticia sebagai solusi
4. SOURCES   → Tunjukkan dari mana bunga tetap ini berasal
5. DEMO      → Tunjukkan cara pakai aplikasi (Faucet → Deposit → Portfolio)
6. TECH      → Sekilas teknologi yang dipakai
7. CLOSING   → Logo + tagline + team
```

### Vibe yang diinginkan

Pikirkan video demo seperti product launch Apple atau Stripe — **clean, confident, tidak berlebihan**. Biarkan produknya yang bicara. Jangan terlalu banyak efek. Yang penting: viewer paham apa yang dilakukan Leticia dalam 30 detik pertama.

---

## Info Teknis untuk Editor

- **Durasi target**: 2-3 menit (maks 3 menit)
- **Tone**: Professional tapi friendly, seperti product demo startup — bukan tutorial teknis
- **Music**: Lo-fi / ambient minimalis, volume rendah di background. Naikkan sedikit saat transisi antar scene
- **Font**: Clean sans-serif (Inter / SF Pro), warna hitam di atas background putih
- **Pacing**: Jangan terlalu cepat. Beri waktu viewer membaca teks di layar (min 2-3 detik per text card)
- **Transisi**: Fade atau smooth cut. Jangan pakai transisi flashy (no wipe, no zoom burst)
- **Color grade**: Clean, minimal. Jangan over-saturate. Tetap konsisten dengan branding Leticia (hitam + putih + pink accent)

## Scene Breakdown

**Visual**: Text card animasi di atas background putih/pink muda

**Text on screen** (muncul satu per satu, timing 2 detik per baris):
```
"You stake sINIT."
"Some weeks you earn 15%."
"Some weeks, 5%."
"What if you could lock in your rate?"
```

**Lalu**: Logo Leticia muncul di tengah dengan tagline di bawahnya

```

Guaranteed yield, zero complexity.
```

> **Catatan editor**: Ini hooks utama. Buat animasi teks yang smooth dan impactful. Bisa pakai kinetic typography style.

### SCENE 5 — Live Demo: Faucet (1:10 - 1:25)

**Visual**: Screen recording browser — Faucet page

**Langkah yang direkam**:
1. Buka halaman Faucet (`/earn/faucet`)
2. Klik "Mint 1,000 INIT"
3. InterwovenKit wallet popup muncul → Approve
4. Balance berubah dari 0 → 1,000

**Text overlay**:
```
"First, grab some free test tokens from the Faucet."
"(This is like getting play money to try the app)"
```

### SCENE 6 — Live Demo: Deposit (1:25 - 1:55)

**Visual**: Screen recording browser — Earn page

**Langkah yang direkam**:
1. Buka halaman Earn (`/earn`)
2. Tampilkan list asset dengan Fixed APR yang terbaca dari smart contract
3. Klik "Deposit" pada INIT (10% APR)
4. Deposit modal muncul — tampilkan:
   - Balance: 1,000 INIT
   - Input amount: 500
   - Estimated profit calculation
   - Yield Source: Enshrined Liquidity
   - Mechanism: LP Staking Rewards
   - Underlying Asset: INIT-USDC LP
5. Klik "Deposit" → Approve popup → Confirm → Deposit popup → Confirm
6. "Deposit Successful" muncul

**Text overlay saat deposit modal terbuka**:
```
"Pick an asset. See your guaranteed rate upfront."
"Check where the yield comes from — fully transparent."
"Approve, deposit, done. That's it."
```

> **Catatan editor**: Ini scene paling penting. Pastikan viewer bisa melihat semua detail di deposit modal dengan jelas. Zoom in ke yield source section. Jangan terlalu cepat — beri waktu 2 detik untuk setiap section.
